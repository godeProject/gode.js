# g;ode.js

Convert gibberish caused by Thai user forgotting to change their keyboard layout.

## Installation

*not to be confuse with "gode" (go+node) package*

via [npm](https://www.npmjs.com/package/gode.js)

```bash
$ npm install gode.js
```

or [yarn](https://yarn.pm/gode.js)

```bash
$ yarn add gode.js
```
